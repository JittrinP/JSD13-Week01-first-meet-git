HTML for Beginners

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a web page using elements enclosed in tags.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

Common HTML Tags

Tag Description
<h1>-<h6>Headings (h1 = largest)
<p>Paragraph
<a>Link (anchor)
<img>Image
<ul>Unordered list
<ol>Ordered list
<li>List item
<div>Block container
<span>Inline container

Key Concepts

  • Tags are enclosed in angle brackets: <tagname>
  • Elements consist of an opening tag, content, and a closing tag: <p>Content</p>
  • Attributes provide extra info: <a href="https://example.com">Link</a>
  • Nesting means placing elements inside other elements

Getting Started

  1. Open any text editor (Notepad, VS Code, etc.)
  2. Write your HTML code
  3. Save the file with a .html extension
  4. Double-click the file to open it in a web browser

Tips for Beginners

  • Always use <!DOCTYPE html> at the top
  • Keep your code indented and readable
  • Use semantic tags like <header>, <nav>, <main>, <footer>
  • Validate your HTML at validator.w3.org
  • Practice by building small projects
